# 날씨 부스트

포켓몬 GO에서 현재 날씨에 맞는 타입의 포켓몬은 **CP가 상승**하고 포획 시 **스타더스트 추가 보상**을 받습니다.

## 날씨별 강화 타입

| 날씨 | 강화 타입 |
|------|---------|
| ☀️ 맑음 / 화창 | 불꽃, 풀, 땅 |
| 🌧️ 비 | 물, 전기, 벌레 |
| ⛅ 구름 조금 | 노말, 바위 |
| ☁️ 흐림 | 페어리, 격투, 독 |
| 🌬️ 바람 | 드래곤, 비행, 에스퍼 |
| ❄️ 눈 | 얼음, 강철 |
| 🌫️ 안개 | 악, 고스트 |

## 레이드 활용 팁

- **보스 약점 타입**이 날씨 부스트를 받으면 → 해당 카운터의 데미지가 **약 20% 추가 상승**
- **보스 자체 타입**이 부스트를 받으면 → 보스 CP가 높아져 포획 난이도 상승
- 날씨 부스트 포켓몬은 야생에서 **Lv.25~35** (통상 Lv.1~30) 범위로 등장
- 레이드 후 포획 시 부스트 적용 → **Lv.25** 기준 (통상 Lv.20)

## 부스트 배율

- 날씨 부스트 받는 타입의 기술 위력: **×1.2**
- STAB(자속 일치)와 중첩 적용 가능 → 최대 **×1.44** (STAB + 날씨)
