# 포켓몬 타입 상성표

포켓몬 GO 레이드/배틀 시 데미지 배율 기준:
- **효과 굉장**: ×1.6 (GO 기준, 메인 시리즈 ×2)
- **효과 별로**: ×0.625 (GO 기준, 메인 시리즈 ×0.5)
- **효과 없음**: ×0.390625 (GO 이중 저항)

> 포켓몬 GO는 메인 시리즈와 배율이 다릅니다. 아래 표는 **유효/무효 관계**를 기준으로 합니다.
